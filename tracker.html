About this project

A live dashboard that automatically tracks Level 6 degree apprenticeships in banking, finance and investment at 12 major employers (HSBC, Barclays, Goldman Sachs, JPMorgan, Morgan Stanley and more).

A scheduled job runs in the cloud three times a day, scrapes the UK Government's official Find an Apprenticeship register, filters out technical roles, flags large-employer opportunities worth a look, and updates this page, so new openings surface the moment they go live.

Built with: HTML/CSS/JavaScript front-end · GitHub Actions for scheduled cloud scraping · GOV.UK data · hosted on GitHub Pages. Your application progress is saved privately in your own browser.
